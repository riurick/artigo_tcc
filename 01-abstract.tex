\begin{abstract}


Este artigo apresenta um estudo sobre usabilidade e testes automatizados no desenvolvimento empírico de software, voltado para a plataforma livre Noosfero, estudo que levantará hipóteses sobre influência dos testes na usabilidade e servirá de base para verificação dessas hipóteses.

\end{abstract}
\vspace{1\baselineskip}

\begin{keywords}
Testes Automatizados, Usabilidade, Métodos Empíricos, Software Livre, Métodos Ágeis
\end{keywords}
